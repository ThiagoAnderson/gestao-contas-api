# Gestão de Contas API

API para controle de despesas pessoais, permitindo registrar, listar e analisar gastos.

## 🚀 Tecnologias
- Java 21
- Spring Boot
- MySQL

## 📦 Funcionalidades (em desenvolvimento)
- Cadastro de despesas
- Listagem de despesas
- Atualização e remoção
- Análise de gastos mensais

## 📁 Estrutura do projeto
- controller/
- service/
- repository/
- model/

## ▶️ Como rodar
1. Clonar o repositório
2. Configurar o banco de dados
3. Rodar a aplicação
